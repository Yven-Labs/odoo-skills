## Code Review Standards (Frontend)

REJECT if:
- OWL component not registered with `registry.category("components").add()`
- `t-raw` used in QWeb for Odoo 17+ (use `t-out`)
- Legacy widget system used in Odoo 17+ (use OWL)
- JS file not using ES modules (`import`/`export`)
- CSS not scoped to Odoo's asset bundle system
