## Code Review Standards (Backend)

REJECT if:
- Model missing `_description`
- Computed field missing `@api.depends`
- `ir.model.access.csv` missing for new model
- `sudo()` used without justification comment
- Raw SQL used without `%s` parameterization
- `name_get()` used in Odoo 17+ (use `_compute_display_name`)
- `attrs="{}"` used in XML for Odoo 17+ (use inline expressions)
