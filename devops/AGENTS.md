## Code Review Standards (DevOps)

- `docker-compose.yml` has hardcoded passwords (use env vars)
- `odoo.conf` committed with real `admin_passwd`
- Commit message not following Conventional Commits
- No `.gitignore` covering `*.pyc`, `__pycache__/`, `.odoo_history`
